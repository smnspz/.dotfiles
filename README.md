# .dotfiles

My favourite settings for bash, vim and vscode.

## Description

This is a `.dotfiles` folder inside `$HOME` in which I store my configuration files for versioning.

### Dependencies

* A unix-like system (MacOS, Linux)
* [GNU Stow](https://www.gnu.org/software/stow/)

### Installing

* Run `git clone git@github.com:smnspz/.dotfiles.git` in your $HOME directory.
